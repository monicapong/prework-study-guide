HTML

  • HyperText Markup Language
  • Handles content, structure, and meaning of each element of the page of a website
  • The following element types are commonly used to add content to pages: text, image, header, list
  • Some common element types that are used to divide and describe a page: meta-related, content division, semantic
  • The head element contains information, or metadata, about the webpage
  • Metadata: data about the webpage that the browser uses to effectively serve our webpage to users
  • Elements included within the head are not visible when a user looks at a webpage
  • The body element represents the visible content shown to the user.
  • !DOCTYPE declaration: tells the browser what document type it should expect when opened
  • lang="en": tells the browser the language for the webpage (in this case, en for English)
  • HTML attribute defines additional infromation about an element
  • title element: defines the title for a webpage
  • style element: defines the CSS styles associated with a website
  • base element: defines the base URL for a webpage
  • link element: connects an external resource to the HTML document
  • meta element: defines metadata such as the character set, description, keyworkds, author, and viewport
  • script element: embeds client-side scripts in the HTML documentation
  • div element: create a division in the page
  • span element: create an incline division
  • Element refers to the entire element from opening tag to closing tag
  • Tag refers only to what appears inside the angle brackets
  • h1, h2, h3, ... elements: the level of heading a given text block represents
  • p element: a parapgraph or block of text
  • ul, ol, li: unordered list, ordered lists, and list items
  • img element: contains information about images that are displayed
  • src element: defines the location of the image
  • br element: creates a line of empty space, or a line break, between two blocks of content
  • a element: (stands for anchor) creates links to the same webpage or other webpages
  • href attribute: points to the URL for the link
  • section element: define a section in a document that contains thematic content

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script.
